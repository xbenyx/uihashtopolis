Agents Status

  • The flashing light indicates the device status.

    ■ Device utilisation is good (> AddConfigThreshold %)

    ■ Device utilisation not ideal (<= AddConfigThreshold %)

    ■ Device utilisation low (<= AddConfigThreshold %)

    ■ Agent is not active

    ■ Agent is active but not working or not providing device data

    ■ Invalid device data from agent or values being 0

    Note: Threshold can be configured in the config section.

  • The flashing light indicates the device status.

    ■ Device temperatures good (<= AddConfigThreshold°C)

    ■ Device temperatures acceptable (<= AddConfigThreshold°C)

    ■ Device temperatures too high (> AddConfigThreshold°C)

    ■ Agent is not active

    ■ Agent is active but not working or not providing device data

    ■ Invalid device data from agent or values being 0

    Note: Threshold can be configured in the config section.

  • The flashing light indicates the device status.

    ■ CPU utilisation is good (> AddConfigThreshold %)

    ■ CPU utilisation not ideal (<= AddConfigThreshold %)

    ■ CPU utilisation low (<= AddConfigThreshold %)

    ■ Agent is not active

    ■ Agent is active but not working or not providing CPU data

    ■ Invalid CPU data from agent or values being 0

    Note: Threshold can be configured in the config section.
